# Hangman V2

This folder contains the second version of a Hangman game implementation in C.

## Contents

- **main.c** - Main game loop and program entry point
- **header.h** - Header file containing function declarations and definitions
- **Debug.c** - Debug version or debugging utilities
- **Debug** - Compiled debug executable
- **main** - Compiled game executable
- **sample.txt** - Sample word list or test data for the game

## Purpose
This is an improved version (V2) of the classic Hangman word-guessing game. Players attempt to guess a hidden word by suggesting letters within a limited number of attempts. The game includes debugging capabilities and uses a sample text file for word selection.

## Features
- Word guessing gameplay
- Letter tracking and validation
- Attempt counter
- Word list from sample.txt
