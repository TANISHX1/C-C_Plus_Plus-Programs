RIFF/WAVE Containerization Visualizer

A RIFF (Resource Interchange File Format) file acts like a set of nested shipping boxes. Every box (called a "chunk") starts with a 4-letter ID and its exact size. This expanded visualization includes the optional LIST container, which is used to store metadata (like Artist and Title) right alongside the audio payload.
