Conversion Process Flow

1. Analog Source

The continuous real-world wave (cream curve). Because computers have finite memory, this infinite line must be discretized into a finite array of numeric values.

2. Sample Rate

Slicing time. The ADC takes vertical "snapshots" (orange stems) at set intervals. Higher frequencies (e.g., 44.1kHz) capture more horizontal detail per second.

3. Bit Depth

Measuring amplitude. The dot must snap to the nearest fixed ruler mark (red grid). Higher bit depths create more marks, minimizing vertical rounding errors.
